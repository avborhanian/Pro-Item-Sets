How this works

This is for those of you wanting to build like that one pro did in that one pro game. Here's how you can use our site to do that:
  1. Find a game you're interested in
  2. Find a player whose build you want to see
  3. Download the build if you like it
  4. The hardest part - move it to the right League folder
    • You just have to move it to the corresponding champ's Recommended folder
    • You can find it at C:\Riot Games\League of Legends\Config\Champions\YourChampGoesHere\Recommended
    • If you don't see your champion's name already listed, you can either create it, or play a custom game with that champ. LoL will autocreate the folders for you
